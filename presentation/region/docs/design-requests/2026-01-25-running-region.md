## 상태별 디자인

### 초기 상태 (검색 전)
- 검색 입력 필드만 표시
- 검색 결과 영역 숨김
- 취소 버튼 (X) 숨김

### 검색 중 상태
- 검색 입력 필드에 텍스트 입력
- 취소 버튼 (X) 표시
- 로딩 인디케이터 표시 (검색 필드 우측 또는 결과 영역 중앙)

### 검색 완료 상태
- 검색 결과 리스트 표시
- **검색어 하이라이팅**: 매칭된 텍스트 Primary 색상(#00EEFF)으로 강조
- 결과 클릭 시 키보드 자동 닫힘

### 검색 결과 없음 상태
- "검색 결과가 없습니다" 메시지 표시
- 폰트: Pretendard Regular 16sp
- 색상: #8F97A3 (BlueGray40)
- 위치: 결과 영역 중앙

### 에러 상태
- "네트워크 오류가 발생했습니다" 메시지
- "다시 시도" 버튼 표시
- 버튼: Primary 색상, 텍스트 버튼 스타일

### 항목 선택 중 상태
- 선택된 항목 로딩 인디케이터 표시
- 다른 항목 클릭 비활성화
