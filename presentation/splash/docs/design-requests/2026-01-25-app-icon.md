# 디자인 작업 요청서 - 앱 아이콘

## 작업 정보
- **작업 유형**: 디자인 구현
- **우선순위**: 높음
- **Figma URL**: https://www.figma.com/design/eaE2AUdQNsllPXAKuHeJrp/RunnersHi?node-id=341-4300

## 디자인 설명
RunnersHi 로고를 사용한 앱 아이콘 업데이트

### 현재 상태
- Foreground: 단순 화살표 아이콘 (#00E5FF)
- Background: #121212

### 변경 사항
- Foreground: RunnersHi 트랙 로고 (ic_logo_runnershi.xml 활용)
- Background: BlueGray90 (#17191C) - 스플래시 배경색과 동일
- Primary 색상 (#00EEFF) 사용

## 기술 요구사항

### Adaptive Icon 구조
```
app/src/main/res/
├── drawable/
│   └── ic_launcher_foreground.xml    # 로고 (108dp, safe zone 66dp)
├── mipmap-anydpi-v26/
│   ├── ic_launcher.xml               # adaptive icon
│   └── ic_launcher_round.xml         # round adaptive icon
└── values/
    └── ic_launcher_background.xml    # 배경색 (#17191C)
```

### 로고 위치
- 108dp x 108dp 캔버스
- 중앙 66dp safe zone 내에 로고 배치
- 여백 고려하여 적절한 크기로 조정
